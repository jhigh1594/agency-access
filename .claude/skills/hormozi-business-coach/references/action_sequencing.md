# Action Sequencing Rules

## Stage-Based Prescriptions

### Below $10K/month
**Fix:** Sales volume ONLY. Nothing else matters.
- Rule of 100: 100 outreaches per day (calls, DMs, emails — pick one)
- Do NOT build a funnel
- Do NOT create content
- Do NOT hire
- Do NOT add products
- Track: daily outreach count, response rate, calls booked, close rate

### $10K-$50K/month
**Fix:** Close rate + Average Order Value. Do not add products.
- Raise prices (test 2x current price)
- Stack the offer (add bonuses that cost you nothing)
- Improve the sales conversation (record and review every call)
- Add an immediate upsell to compress payback period
- Track: close rate, AOV, payback period, refund rate

### $50K-$100K/month
**Fix:** Delivery. Systematize fulfillment to remove yourself.
- Document every delivery process
- Hire first fulfillment person
- Create client onboarding system
- Measure: NPS, retention rate, time to result
- Stop doing ANY delivery yourself

### $100K+/month
**Fix:** Team and systems. Build the machine.
- Hire sales (you should NOT be closing)
- Build management layer
- Implement dashboards for all key metrics
- Focus on: LTV optimization, referral systems, expansion revenue

## The "More, Better, New" Protocol (Always in This Order)

1. **Do MORE** of what's already working. Max volume.
2. **Do it BETTER.** Optimize conversion at each step.
3. **Do something NEW.** Only when 1 and 2 are maxed. This is almost never the answer below $1M.

## Universal Rule
If stuck at any plateau: raise prices FIRST, increase volume SECOND. Price is always the first lever.
