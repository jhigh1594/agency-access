# Create Your First Request

Your first milestone is simple: create one live request for one real client.

## What to prepare

- The client name you want to onboard first.
- The platforms you need access to.
- The access level the client should approve.

## Fastest setup path

1. Sign in and open the dashboard.
2. Choose **New access request**.
3. Select the client.
4. Pick the platforms and permissions you need.
5. Generate the request link.

## Keep the first request narrow

Do not try to configure every agency detail before sending the first request.

- Start with the platforms blocking current work.
- Use the default branding if you have not finished customizing it.
- Focus on shipping one clean request, then refine the workflow after activation.

## What good looks like

- The request is live.
- You have a secure link ready to send.
- The client can complete authorization without extra instructions from your team.

## Next step

Once the request exists, move to [Send your link](/requests-and-links/send-your-link).
