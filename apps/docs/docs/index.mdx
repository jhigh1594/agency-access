## Start Here

- New to the product? Go straight to [Create your first request](/getting-started/create-your-first-request).
- Waiting on a client? Use [Send your link](/requests-and-links/send-your-link) and [Check request status](/requests-and-links/check-request-status).
- Automating downstream work? Start with [Webhooks](/automation/webhooks).
- Need platform-specific walkthroughs? Jump to our public guides for [Google Ads access](https://www.authhub.co/blog/google-ads-access-agency) and [Meta Ads access](https://www.authhub.co/blog/how-to-get-meta-ads-access-from-clients).
