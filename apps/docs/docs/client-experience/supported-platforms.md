# Supported Platforms Overview

AuthHub supports a mix of ads, analytics, ecommerce, and email marketing platforms so agencies can keep onboarding inside one system.

## Core platform groups

- Google
- Meta
- LinkedIn
- TikTok
- Pinterest
- Shopify
- Klaviyo
- Kit
- Beehiiv
- Mailchimp
- Zapier

## When to use a platform-specific guide

Send clients to a guide when a platform has non-obvious steps outside the AuthHub authorization flow, or when your team needs to explain platform terminology before the client starts.

Useful public guides:

- [Google Ads access guide](https://www.authhub.co/blog/google-ads-access-agency)
- [Meta Ads access guide](https://www.authhub.co/blog/how-to-get-meta-ads-access-from-clients)
- [GA4 access guide](https://www.authhub.co/blog/ga4-access-agencies)

## Internal operating rule

If a client asks for platform-specific setup details, do not improvise from memory. Use the documented guide or update the guide before repeating the same answer manually.
