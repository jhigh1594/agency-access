# Agency Branding Basics

Branding matters because it tells the client the request is legitimate and expected. It should support trust, not block speed.

## Prioritize these first

- Agency name.
- Logo.
- Brand colors that match your client-facing materials.

## What to avoid

- Delaying the first live request for visual polish.
- Changing branding so frequently that clients see inconsistent experiences.
- Using branding that makes support emails and authorization pages feel disconnected.

## Recommendation

Launch with a clean branded baseline, then refine once your team has repeated the process enough to understand what clients actually question.
