# Simple Skill Template

Use this template for straightforward skills that fit entirely in a single SKILL.md file (typically under ~2000 words).

## When to Use This Template

- Single, focused purpose
- Instructions fit in one file (< 2000 words)
- No code execution needed
- All content is typically needed together

## Template

```markdown
---
name: your-skill-name
description: A clear, specific description of what the skill does and when it should be used.
---

# Skill Name

## Purpose

[Describe what this skill enables Claude to do]

## When to Use This Skill

[Describe the scenarios or tasks that should trigger this skill]

## Instructions

[Provide detailed instructions for Claude on how to perform the task]

## Examples

[Include examples of how to use this skill or what outputs to expect]

## Notes

[Any additional context, limitations, or important considerations]
```

## Customization Tips

1. **Name**: Use kebab-case, be descriptive but concise
2. **Description**: Be specific about when the skill should trigger
3. **Instructions**: Be clear and actionable
4. **Examples**: Include real examples to clarify usage
