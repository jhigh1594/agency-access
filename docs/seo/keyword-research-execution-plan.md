## Content Format Standards

### Troubleshooting Articles
- **Structure**: Problem → Why it happens → Step-by-step fixes → Prevention
- **Visuals**: Screenshots, error message examples, flow diagrams
- **Length**: 800-1,200 words
- **Internal links**: To relevant platform guides

### How-To Guides
- **Structure**: Prerequisites → Agency-side steps → Client-side steps → Verification
- **Visuals**: Screenshots at each step, annotated UI elements
- **Length**: 1,200-1,800 words
- **Internal links**: To related troubleshooting, tool comparison

### Best Practices Roundups
- **Structure**: Why it matters → X practices → Examples → Tools
- **Visuals**: Process diagrams, checklists, comparison tables
- **Length**: 1,500-2,500 words
- **Internal links**: Deep links to specific practice articles

### Comparison Articles
- **Structure**: What to compare → Feature comparison → When to choose what
- **Visuals**: Comparison table, decision tree
- **Length**: 1,000-1,500 words
- **Internal links**: To individual product pages

## Content Production Workflow

1. **Keyword validation** → Confirm volume, check SERP
2. **Cannibalization check** → Verify no existing content targets term
3. **Competitor research** → What ranks? How to differentiate?
4. **Outline creation** → Structure based on intent type
5. **Draft creation** → Write with agency-specific examples
6. **Design integration** → Add brutalist elements, screenshots
7. **Internal linking** → Connect to cluster pages
8. **Publish + promote** → Share in agency communities, social
9. **Track performance** → GSC weekly, adjust strategy

## Risks & Mitigations

| Risk | Mitigation |
|------|------------|
| Volumes overestimated | Validate with tools before prioritizing |
| Cannibalization missed | Run `site:` queries for each keyword |
| Content doesn't rank | Focus on long-tail first; build authority gradually |
| Competitors have better resources | Differentiate with design, specific agency expertise, brutalist UX |
| Algorithm updates affect rankings | Diversify traffic sources; build email list via gated content |
